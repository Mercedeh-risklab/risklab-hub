# Excel Learning – Week 1 (Simple Notes)

## Overview
This week I focused on improving my practical Excel skills, especially in areas that help keep large files lighter and easier to work with. I also reviewed basic function categories and a few useful techniques for cleaning and organizing data.

## Functions I Learned (Basic Categories)

### ✔ Lookup Functions
- VLOOKUP  
- XLOOKUP  
- INDEX + MATCH  
Useful for searching values and linking tables together.

### ✔ Logical Functions
- IF  
- AND / OR  
Helpful for creating simple rules and filtering conditions.

### ✔ Text Functions
- LEFT / RIGHT  
- MID  
- TRIM  
Useful for cleaning messy text data.

### ✔ Basic Statistical Functions
- AVERAGE, COUNT, MAX, MIN  
- COUNTIF, SUMIF  
Good for quick analysis of structured datasets.

---

## Practical Excel Skills Practiced

### ✔ File Performance (How to Keep Excel Lightweight)
- Using **Tables** instead of normal ranges  
- Removing unnecessary formulas  
- Clearing heavy formatting  
- Splitting raw data into separate sheets or files  
- Using **Paste Values** after calculations  
These techniques help avoid slow calculation times and heavy files.
